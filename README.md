Password Strength Checker

Description

A Python-based Password Strength Checker that evaluates password security using:

- Password length validation
- Uppercase letter detection
- Number detection
- Special character detection
- Password scoring system

Technologies Used

- Python
- Colorama
- Regular Expressions (Regex)

Features

- Strong/Weak password detection
- Password score meter
- Colored terminal output
